# 블랙잭 앱 프로젝트 메모리

최종 업데이트: 2026-04-05 (Asia/Seoul)

## 제품 목표

- 사내에서 플레이 가능한 멀티플레이 블랙잭 데스크톱 앱 제공
- 기술 스택: ElectronJS(macOS/Windows), Supabase, TailwindCSS
- 무인증 간편 입장: 앱 실행 후 닉네임만 입력

## 핵심 기능 요구사항

- 최대 5인 플레이(딜러는 컴퓨터)
- 로비:
  - 개설된 방 목록 조회
  - `게임 생성하기` 버튼
- 방 생성:
  - 방 이름
  - 방 비밀번호(선택)
  - 최대 입장 인원
  - 승리 조건 금액(N$)
  - 참가자 시작 자금(10$~1000$)
  - 게임 시작 시 자동 기본 베팅(1$~100$)
- 방 내부:
  - 참가자 리스트
  - 채팅
  - 방장 표시 아이콘
  - `READY` 상태 표시
  - 전원 READY 후 방장이 시작
- 게임:
  - 시작 시 자동 기본 베팅
  - 행동: 표준 블랙잭 기준(현재 구현: Hit / Stand / Double)
  - 연결 끊김 시 자동 패배 처리
  - 라운드 종료 후 우승자 미확정 시 다음 라운드 자동 시작
  - 보유금 0$ 이하 시 탈락
  - 최종 1인 생존 또는 목표 금액 선점 시 우승

## 업데이트 요구사항

- 기존 설치자(예: 1.0.0)가 앱 실행 시 신규 버전(예: 1.0.1) 패치 다운로드 후 업그레이드 가능해야 함
- 플랫폼: macOS + Windows

## 현재 상태

- Electron + React + TypeScript 템플릿 초기 상태
- `@supabase/supabase-js`, `electron-updater` 의존성 이미 설치됨

## 확정된 정책 (2026-04-05)

- 게임 룰: 블랙잭 표준 룰로 전환
- 방 최대 인원: 1~5명(추후 확장 고려)
- 연결 끊김: 30초 유예 후 패배 처리
- 닉네임: 방 내 중복 불가
- 닉네임 변경: 로비에서 언제든 변경 가능
- 우승 판정: 라운드 종료 후 판정
- 자동 업데이트 호스팅: 사내 서버
- 로그 보관: 채팅 로그 제외, 게임 진행 로그만 저장
